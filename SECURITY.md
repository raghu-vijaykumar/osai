# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest stable release | Yes |
| Beta releases | No |
| Nightly builds | No |

## Reporting a Vulnerability

Report vulnerabilities by emailing `security@osai.app` (placeholder — update before launch).

Do not file a public issue for security vulnerabilities.

You will receive a response within 48 hours. If the issue is confirmed, a patch will be released within 14 days, and the reporter will be credited (if desired).

## Disclosure Policy

- Vulnerabilities are disclosed 30 days after a fix is released
- Critical vulnerabilities may warrant an immediate out-of-band release
- We follow coordinated disclosure — no public details before the fix ships
